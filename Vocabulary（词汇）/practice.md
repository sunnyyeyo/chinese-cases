词汇拓展练习
说明：根据《国际中文教育中文水平等级标准》语言量化指标总表，初级（一至三级）的词汇量要求分别为：一级500词，二级1272词（新增772词），三级2245词（新增973词）。以下是为初级1-3级设计的词汇练习题，涵盖词根、词缀、同义词、反义词、搭配等多种形式，旨在帮助学生扩大词汇量。
初级一级词汇练习题（共500词）
一、词根/词缀扩展
1.	“子”后缀：很多名词以“子”结尾。请写出以下词语的完整形式：

包__ → ______
本__ → ______
2.	“儿”化音：一些名词可以加“儿”变成更口语化的词。请写出以下词语的“儿”化形式：
小孩 → ______
好玩 → ______
一会儿 → ______
哪儿 → ______
二、同义词连线
将左边词语和右边意思相近的词语连起来。
喜欢       父亲
经常       漂亮
好看       爱
爸爸       常常
三、反义词填空
选择正确的反义词填入句子中。
（大 / 小， 多 / 少， 好 / 坏， 来 / 去）
1.	这个苹果很______，那个苹果很______。
2.	教室里的学生很______，图书馆里的人很______。
3.	今天天气很______，昨天天气很______。
4.	他刚从北京______，明天要______上海。
四、词语搭配

3.	______ 音乐 （听/吃）

初级二级词汇练习题（新增772词）
一、词根扩展（动词+结果/趋向）
很多动词可以加“到”、“见”、“完”等表示结果，或加“来”、“去”、“上”、“下”等表示趋向。请用这些补语填空。
（到、见、完、来、去）
1.	我看______一本书了。（看完）
2.	他找______了他的手机。（找到）
3.	请听______我说的内容。（听见）
4.	妈妈买______了很多水果。（回来）
5.	弟弟跑______房间了。（进去）
二、同义词替换
用括号里提供的更高级的同义词替换句中划线词语。
1.	这个菜很好吃。（美味）→ 这个菜很______。
2.	他很快地跑开了。（迅速）→ 他______地跑开了。
3.	我对这个电影有兴趣。（感兴趣）→ 我对这个电影______。
4.	我们需要更多时间。（足够）→ 我们需要______的时间。
三、反义词配对
找出以下词语的反义词，并组成配对。
（安全、容易、简单、危险、复杂、困难）
•	安全 — ______
•	简单 — ______
•	容易 — ______
四、词语辨析（近义词选择）
选择最合适的词语填空。
（参观 / 访问， 帮助 / 帮忙， 忽然 / 突然）
1.	明天我们要去______博物馆。（参观）
2.	他可以______你解决这个问题。（帮助）
3.	天气______变冷了，你要多穿衣服。（突然）
五、成语/固定结构入门
将下列四字词语与它的解释连线。
各种各样              形容数量很多
越来越……            表示种类多
一边……一边……      表示时间长了结果怎么样
很多很多              表示两个动作同时做

二、同义词进阶（书面语替换）
用括号中更书面的同义词替换句中划线词。
1.	他的成绩很好。（优秀）→ 他的成绩很______。

四、词语搭配与辨析
1.	搭配：选择动词与名词正确搭配。
